Mapty App

Mapty is an application written in JavaScript that allows the user as a recorder of exercises performed. To begin, you must allow your browser to access the location. Then you can click on the map at the location you want to record. The completed form is approved with the enter button. The application also allows you to edit, delete and sort exercises. The exercise log is stored in the browser's local memory.

Mapty App

Restaurant Web

Restaurant web is an example of a restaurant website with a banner that scrolls across the width of the screen, this site has a scrolling slider with a function to scroll automatically every specified time or manually. This site has the features of a fully responsive web site. The site is built using HTML, SCSS, SWIPER.

Pig Game

Pig Game is a game in which two players draw dice, thus scoring points. When the user draws one eye, he loses a turn and the turn goes to his opponent. Whoever collects 100 points first wins. The functionality of the application is provided by JavaScript
